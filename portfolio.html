Get Out!

Get Out! was a simple-one page site created using Javascript, Bulma and leveraged the API's of OpenWeather and Eventbrite to allow travelers to a given city to see the weather five days in advance and to search for given activities in that location.

PDi Concept Site

This site is an under development concept created for Promotional Development, a Display and Advertising Firm in Brooklyn, NY. The site was created using React and highlights the clean look which PDi is known for in the display industry.

Interested?

Interested is a multi-page dating app created using React, Material UI, MongoDB and Firebase. A user has the ability to sign in and create their own profile, then contact other users via E-mail.
