Get Out!

Get Out! was a simple-one page site created using Javascript, Bulma and leveraged the API's of OpenWeather and Eventbrite to allow travelers to a given city to see the weather five days in advance and to search for given activities in that location.

Interested?

Interested is a an under development multi-page dating app created using React, Material UI, MongoDB and Firebase. A user has the ability to sign in and create their own profile, then contact other users via E-mail. The link leads to the GitHub at this time.

New Jersey Devils Trivia

New Jersey Devils Trivia is a basic trivia app created with HTML/CSS/JQuery. The end user will start the quiz and then have 60 seconds to answer the Devils Trivia questions!
